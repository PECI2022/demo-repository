
# Setup

Create virtual environment (venv)

```bash
python3 -m venv venv
```

Activate venv

```bash
source venv/bin/activate
```

Install dependencies

```bash
pip install -r requirements.txt
```

# Run

Start Backend server

```bash
cd Backend
python3 server.py
```

Start Frontend server

```bash
cd Frontend
python3 server.py
```

# Open

Start a browser on http://127.0.0.1:8080/
